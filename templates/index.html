Some Key Points On How This Website Works

Datasets represent a set of sequencing files coming from a particular Illumina library.

By defining attributes such as chain and species when adding a dataset, we can populate new analyses with the most appropriate default settings. This will also enable auto-analysis of data when it becomes available.

When running an analysis on a particular dataset, files coming from the same chain need to be pre-processed for overlap alignments and the consensus sequences taken into analysis. You can also pre-run the overlap alignment with alternative settings if you like - the overlaps from the freshest alignment will be analyzed.



Bear in mind this is in pre-release development, but we're heading toward multi-dataset comparisons to generate the plots we all want and need.
